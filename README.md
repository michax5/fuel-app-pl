# fuel-app-pl


# Aplikacja paliwowa (analiza cen ropy i paliw)

## 📊 Opis projektu

Projekt analizuje wpływ cen ropy na ceny paliw w Polsce,
w tym na maksymalne ceny paliw ustalane administracyjnie.

Aplikacja łączy dane o cenie ropy Brent oraz kursie USD/PLN,
aby oszacować koszt surowca w złotówkach i jego wpływ na ceny paliw.

---

## 🎯 Cel projektu

Celem projektu jest:
- analiza zależności między cenami ropy a cenami paliw
- identyfikacja opóźnienia (lag) zmian cen
- wizualizacja wyników w interaktywnym dashboardzie
- stworzenie prostego sygnału decyzyjnego (kup teraz / poczekaj)

---

## 🛠 Technologie

- Python (Pandas, NumPy)
- API (yfinance – ropa, NBP – kurs USD)
- Streamlit (dashboard)
- Analiza danych
- Wizualizacja danych

---

## 📈 Zakres projektu

- pobieranie danych z API
- przetwarzanie i czyszczenie danych
- analiza danych czasowych
- identyfikacja opóźnienia (lag)
- wizualizacja w aplikacji Streamlit
- generowanie sygnału decyzyjnego

---
